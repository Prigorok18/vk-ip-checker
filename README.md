# vk-ip-checker
### vk-ip-checker создан для нахождения "хороших ip" у вк клауд

## ⚡ Быстрая установка

Скопируйте и выполните в терминале:

```bash
curl -fsSL https://raw.githubusercontent.com/Prigorok18/vk-ip-checker/main/install.sh | sudo bash
```
## ✅ После установки

Скопируйте и выполните в терминале:
```bash
vk-ip-checker
```
